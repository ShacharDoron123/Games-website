## 🕹️ About the Project

This project is a simple and clean website that lets users play different games directly in their browser — no downloads needed.

It was built as a learning project to practice:

* JavaScript
* Game logic
* UI design
* Web development

## 📦 Installation

If you want to run the project locally:

```bash
git clone https://github.com/ShacharDoron123/Games-website.git
cd Games-website
```

Then open `index.html` in your browser.

---

## 🌍 Deployment (GitHub Pages)

1. Go to repository settings
2. Scroll to **Pages**
3. Select branch: `main`
4. Save
